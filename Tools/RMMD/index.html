如何使用？

  1. 支持的加密文件的后缀有 .rpgmvp / .rpgmvm / .rpgmvo / .png_ / .ogg_ / .m4a_ ，否则无法使用此脚本
    如果你需要加密文件，它们通常有一个文件扩展名，如 "png" =)
  2. 获取 加密/解密密钥
    New: PNG文件 （如 .rpgmvp / .png_ 图像文件）无需密钥, 除非 你需要重新加密文件。 点我前往转换(没有密钥的)图像
    • 上传游戏的 System.json 文件后，点击“检测”运行检测代码
    • 如果检测不起作用，则会弹出进一步的说明
  3. 选择要解密/加密的文件 - 可同时选择多个文件！
  4. 点击解密或加密按钮
  5. 如果您点击文件名旁边的“预览”，则可以预览解密的文件。（加密的文件除外）
  6. 您现在可以通过点击“保存”来保存它们。 =)
    • 您还可以通过单击“全部另存为.zip文件”将文件列表中的所有文件打包为.zip格式！
