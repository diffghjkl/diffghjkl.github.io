利用 GitHub + jsdelivr 搭建免费图床
发表于：2022-05-15 | 分类： 技术教程

前言

截止至2022/05/25，国内仅部分地区可正常访问 jsdelivr

之前写博客的时候发现个问题：

用网上免费的图床不是图片总被删就是速度不快，还具有不稳定的因素…

（头像图片被删我是很不理解的）

之后，

在无意之中发现利用 Github 的存储库 + jsdelivr 提供的CDN服务 可以搭建图床~

那么，开始教程吧！

（注：格式部分是为了防止图片失效）

准备

  • 一个 GitHub 账号
  • 一台电脑
  • 安装 PicGo

如果你没有电脑，用手机也不是不行…
